# AC-Dino-Runner
## Dino Runner es un videojuego donde el objetivo principal es que el dinosaurio corra y pueda pasar por todos los obstaculos

## el dinosaurio tiene el podeer del escudo que lo protege de los obstaculos
## el dinosaurio tiene el poder del martillo qu destrulle todos los obstaculos
## se aumenta una vida cada 200 puntos teniendo un limite de 5 vidas 
## tiene un contador de cuantas muertes tubo el dinosaurio
## tiene un contador que te muestra cuel fue tu ultimo puntaje
